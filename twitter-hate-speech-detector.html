The Twitter Hate Speech Detector is an independent NLP research project that applies a fine-tuned BERT model to classify hate speech in social media posts. Using PyTorch and Hugging Face's pretrained BERT architecture, the model attains 82% validation accuracy across a dataset of over 10,000 tweets, targeting racist and sexist content. The project connects directly to Shalini's broader interest in using NLP to counter online harm, related to the Future of Free Speech Project and her technical skills in machine learning.

Motivation

Hate speech detection is a foundational problem in content moderation. Unlike rule-based filters, ML classifiers can generalize to novel phrasings and detect intent from context. This project was motivated by the same questions driving the Future of Free Speech Project — how can computational tools help reduce harm online, and what are the limits of automated moderation?

Model Architecture

The model uses Hugging Face's pretrained BERT (Bidirectional Encoder Representations from Transformers) as a base, fine-tuned on a labeled Twitter dataset for binary classification of hate speech. BERT's bidirectional attention mechanism makes it well-suited for understanding tweet context, slang, and implicit hostility that simpler bag-of-words approaches miss.

Dataset

The training dataset consists of over 10,000 labeled tweets, balanced across racist, sexist, and neutral classes. Data collection, management, and structuring were done with custom preprocessing pipelines for text classification tasks.

Data Validation

The project applied rigorous validation methods including cross-validation, bias detection across demographic subgroups, and dataset quality assessment to ensure model reliability. A key concern was whether the classifier unfairly flagged language used by the very communities it aimed to protect — a problem well-documented in hate speech detection literature.

Results

The model achieved 82% validation accuracy on the held-out test set.

Limitations and Open Problems

Binary classification into "hate speech / not hate speech" misses the nuance of real online abuse: irony, reclaimed language, context-dependent slurs, and coordinated harassment that is individually benign. Current work focuses on expanding the target classes of abuse beyond racist and sexist content and deploying the full pipeline to production.

Open Source Contributions

Portions of the model training framework are being published to GitHub to contribute to the open-source NLP community.
