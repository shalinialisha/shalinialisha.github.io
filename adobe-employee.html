Frame.io and Adobe

Frame.io is Adobe's cloud-based video collaboration platform, used by professional filmmakers, editors, and studios to review, annotate, and approve video content. The Developer Platform API team owns the external-facing REST API that enables third-party integrations, automation workflows, and programmatic access to Frame.io's video asset management system.

Day-to-Day Engineering

Shalini's engineering work spans the stack. On the frontend, she builds JavaScript features for the developer-facing API experience, including webhook integrations that let developers subscribe to real-time Frame.io events.

Self-Automating Zapier Framework

Shalini designed and built a self-automating integration framework for the Frame.io Zapier V4 integration. The system monitors the Frame.io OpenAPI spec for changes, automatically generates updated Zapier action and trigger code when the API changes, and opens a pull request in the Zapier repository — eliminating the manual effort of keeping the integration in sync with API evolution.

OpenAPI Diff Detection Pipeline

The automation is powered by a GitHub Actions workflow that diffs the OpenAPI spec on each API change, identifies which endpoints have been added, modified, or deprecated, and triggers code generation accordingly.
