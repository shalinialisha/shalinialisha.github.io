Shalini joined Frame.io (an Adobe company) as a full-time Software Engineer after completing her internship and graduating from Vanderbilt. She is one of three engineers who co-architected the Frame.io CLI and MCP server — building both from the ground up in roughly a month. She also designed a self-automating Zapier framework that has significantly reduced manual workflow overhead across the team. See also Technical Skills and Community Involvement.

CLI and MCP Server

As one of three engineers on the project, Shalini co-architected the Frame.io command-line interface and Model Context Protocol server. The project went from design to delivery in approximately one month — a pace that required tight coordination, rapid iteration on API design, and close collaboration with product teams to ensure the CLI met real user workflows.

The MCP server enables AI tools and agents to interact with Frame.io programmatically, extending the platform's capabilities into emerging AI-powered workflows.

Self-Automating Zapier Framework

Shalini designed and built a self-automating Zapier framework that generates and maintains workflow automations without requiring manual configuration for each new integration. The system has reduced manual workflow overhead for the team and serves as infrastructure for connecting Frame.io to third-party tools.

Technical Stack

Engineering work at Frame.io is done primarily in Elixir, a functional programming language built on the Erlang VM. Shalini also works with REST API design, OpenAPI, and GitHub Actions for CI/CD automation. See Technical Skills for a full inventory.

Community at Adobe

Alongside her engineering work, Shalini is an active member of HOLA — Adobe's Hispanic and Latino Organization for Leadership and Advancement — where she helps plan events educating the broader Adobe community on Latino and Hispanic history and culture. See Community Involvement for more detail.
